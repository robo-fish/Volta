Teilschaltungen sind wiederverwendbare Schaltungsfragmente. In Volta sind sie als spezielle Schaltungsdokumente realisiert. Teilschaltungen können in die Komponentensammlung geladen werden, wo sie wie Komponentenmodelle behandelt werden.

Teilschaltungen erstellen

Folgen Sie diesen Schritten um eine Volta Teilschaltung, ausgehend von einem Schaltungsdiagramm oder von einer SPICE Teilschaltung, zu erstellen:

  1. Teilschaltungs-Editor öffnen
  2. Teilschaltung aktivieren
  3. Kennung (Name) eingeben
  4. Optional Herstellerkennung eingeben
  5. Anzahl der externen Anschlußstifte angeben
  6. Verbindungsknoten für Anschlußstifte angeben
  7. Schaltungsdiagramm erfassen
Abb. 1: Erstellen einer Teilschaltung

Bei Teilschaltungen die auf einer SPICE-Netlist basieren wird der letzte Schritt nicht ausgeführt.

In Schritt Nr. 5 wählt man die Anzahl der Anschlußstifte für das DIP-Symbol ("Dual In-line Package") der Teilschaltung aus. Derzeit sind nur DIP-Symbole verfügbar.

Die Zuweisung von externen Anschlüssen auf interne Knoten in Schritt Nr. 6 definiert die Schnittstelle zu dem Knotennetzen der Teilschaltung. Ohne diese Schnittstelle ist die Teilschaltung für den Simulator nicht zugänglich. Vergewissern Sie sich, dass die Knotennamen, die Sie eingeben, tatsächlich in der Netlist vorhanden sind. Für Anschlüße die nicht benutzt werden lassen Sie den Knotennamen leer.

Tipp: Bauen Sie Testschaltungen um zu prüfen ob ihre Teilschaltung wie erwartet funktioniert.

Teilschaltungen anwenden

Damit man die gebaute Teilschaltung verwenden kann muss sie in das Teilschaltungsverzeichnis abgelegt werden. Die Komponentensammlung aktualisiert daraufhin automatisch die Liste der Teilschaltungen ohne dass ein Neustart notwendig ist.

Das Teilschaltungsverzeichnis kann im Finder angezeigt werden indem man auf das Verzeichnissymbol (siehe Abb. 2) klickt.

Teilschaltungen können direkt in ein Schaltungsdiagramm oder in eine Palettengruppe gezogen werden um sie bequem aus der Komponentenleiste heraus zu benutzen.
