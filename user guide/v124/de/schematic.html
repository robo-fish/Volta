Der Schaltungsdiagramm-Editor (Abb. 1) besteht aus einer Komponentenleiste (oder Palettenleiste), einem Infopanel und einem Diagrammfeld zum Zusammensetzen von Komponenten.

Abb. 1: Schaltungsdiagramm-Editor

Komponentenleiste

Die Komponententleiste des Diagramm-Editors befindet sich über dem Diagrammfeld und zeigt eine Reihe von Komponenten an. Man kann damit bequem auf häufig benutzte Komponenten zugreifen ohne das Fenster für die Komponentensammlung öffnen zu müssen. Änderungen am Inhalt der Komponentenleiste können allerdings nur im Komponentenfenster gemacht werden.

Ziehen Sie Komponenten einfach aus der Leiste runter um sie in das Diagramm einzufügen (Abb. 2).

Abb. 2: Komponente aus der Leiste ziehen

The Komponentenpalette in der Komponentensammlung kann aus mehreren Komponentengruppen bestehen. Die Komponentenleiste zeigt jedoch nur Komponenten einer einzigen Gruppe an. Über das Menü links oben können Sie die Gruppe auswählen (Abb. 3).

Abb. 3: Komponentengruppe auswählen

Mit "Bearbeiten..." gelangen Sie zur Komponentensammlung.

Die Komponentenleiste kann mit dem Symbolleistenknopf "Palette" aus- und wieder eingeblendet werden.

Das Infopanel

Im Infopanel kann man selektierte Komponenten bearbeiten. Man kann Komponenten rotieren und spiegeln, Komponentenmodelle auswählen und Werte für die Attribute eingeben. Bei Einzelselektion kann man den Namen einer Komponente ändern falls dieser Name nicht schon vergeben ist. Bei Mehrfachselektion kann man nur Attribute bearbeiten die alle selektierten Komponenten gemeinsam haben.

Der Name der selektierten Komponente wird ganz oben angezeigt. Mit einem Klick kann man ihn sofort bearbeiten. Unterhalb des Namensfeldes befinden sich die Knöpfe zum Platzieren des Namens, zum Rotieren und zum Spiegeln der Komponente, weiter drunter das Menü zur Auswahl alternativer Komponentenmodelle und als Letztes die Tabelle der Komponentenattribute mit ihren Werten. Ein Attributenwert kann mit einem Doppelklick auf das Feld neben dem Attributennamen bearbeitet werden.

Abb. 4: Das Infopanel

Bei Schaltungskomponenten sind die angezeigten Attribute SPICE-Attribute. Deren Beschreibungen finden Sie im Ngspice Benutzerhandbuch.

Das Infopanel kann durch drücken des Symbolleistenknopfes "Info" aus- und wieder eingeblendet werden.

Das Infopanel lässt sich an die Ecken des Diagrammfeldes haften.

Komponenten verdrahten

Zwei Schaltungskomponenten können miteinander verbunden werden indem man eine Verbindungslinie von einem Anschlußpunkt (Stift) zum Anderen zieht. Jede elektrische Schaltungskomponente hat mindestens einen Anschlußpunkt. Ein Anschlußpunkt wird in Rot angezeigt wenn sich der Mauszeiger darüber befindet. Einen Anschlußpunkt kann man nur ein Mal verbinden. Verwenden Sie Knotenelemente mit jeweils 4 Anschlußpunkten um Verbindungen zusammenzuführen. Um eine Verbindung zu entfernen ziehen Sie die Verbindungslinie einfach von einem der beiden Enden ab.

Abb. 5: Komponenten verbinden

Eine Verbindung wird als gerade Linie erzeugt. Durch ziehen an geraden Linienabschnitten ist es möglich Gelenke einzufügen (siehe Abb. 6). Zum Entfernen eines Gelenks schiebt man das Gelenk auf die (imaginäre) gerade Linie zwischen den beiden Nachbarpunkten des Gelenks. Gelenke werden in Grün angezeigt wenn sich der Mauszeiger darüber befindet.

Diagramm vergrößern

Das Schaltungsdiagramm kann bis zur doppelten Standardhöhe und Breite vergrößert werden. Die Vergrößerung ist für ein angenehmeres Arbeiten auf kleinen, hochauflösenden Bildschirmen gedacht. Der Vergrößerungsfaktor kann mittels dem Regler in der Toolbar (Abb. 7) oder durch eine Kneifgeste im Trackpad verändert werden.
